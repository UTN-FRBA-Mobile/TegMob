## Detalles del proyecto

### Descripción
    Se trata de una app que permite jugar al Teg de forma online entre varios jugadores

### Alcance
#### Incluye
- Logueo del usuario
- Creación de partida online
- Adhesión a partida online
- Juego online

##### No incluye
- Creación de partida local
- Juego local
